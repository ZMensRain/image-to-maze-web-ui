This is a web wrapper of [another project](https://github.com/ZMensRain/image-to-maze) of mine written in go and in this project compiled to webassembly.

## Demo

https://image-to-maze-web-ui.vercel.app/

# Build instructions

## Requirements

- go
- pnpm
- node

## Instructions

1. `cd ./src/lib/go`
2. `go get`
3. `cd ../../../`
4. `pnpm build`
